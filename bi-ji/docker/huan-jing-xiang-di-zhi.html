# docker更换镜像仓库

  1. 进入目录
    cd /etc/docker

  2. 打开daemon.json，没有就新建一个
    vim daemon.json

  3. 配置以下内容

{

    "registry-mirrors":["https://almtd3fa.mirror.aliyuncs.com"]      
}
